# 电子大刘 (AI-DaLiu)

AI 科幻创作与短片制作引擎，基于 [Claude Code](https://claude.ai/claude-code) Skills 构建。

## 这是什么

"电子大刘"是一套 Claude Code Skills，能够：

1. **以刘慈欣风格创作科幻小说** — 基于对大刘 26 部重要作品的逐篇精读与分析，学习其构思方法、叙事结构、场景写作和人物塑造的内核
2. **从文本自动生成 AI 短片** — 全流程自动化：文本分析 → 分镜脚本 → AI 图片/视频生成 → TTS 旁白 → 字幕 → BGM → 最终组装
3. **三体同人创作** — 在三体世界观内创作同人小说，严格遵守原作的物理规则和文明设定

## Skills 一览

| Skill | 调用方式 | 功能 |

## 短片制作技术栈

- **图片生成**: 火山方舟 Seedream 5.0
- **视频生成**: 火山方舟 Seedance（2.0 → 1.5 Pro → 1.0 Pro 逐级降级）
- **TTS 旁白**: edge-tts + SSML 情感控制
- **视频组装**: FFmpeg
- **BGM**: Incompetech 免费曲库（CC BY 4.0）

## 使用方式

1. 安装 [Claude Code](https://claude.ai/claude-code)
2. 克隆本仓库，在项目目录下启动 Claude Code
3. 使用上述 Skill 命令开始创作

短片制作还需要：
- 火山方舟 API Key（设置环境变量 `ARK_API_KEY`）
- FFmpeg
- Python + edge-tts（`pip install edge-tts`）
