# 三体人物分析库

本 skill 汇总了对《三体》三部曲主要人物的系统分析。每个人物是一个独立的参考文件（references/ 下），基于对原文的逐字精读提炼而成，可在需要时按需读取。

## 使用方式

根据任务涉及的人物，读取对应的 references/ 文件。**不要一次性读全部**——只加载当前任务需要的人物。

## 何时使用本 skill

1. **创作校验**：在三体宇宙内写作（同人、日记提及、短片脚本等）涉及上述人物时，先读对应分析，确保性格、决策逻辑、台词语气一致。
2. **角色分析讨论**：用户直接询问某个人物的动机、选择、象征意义。
3. **创作风险提示**：每个 references 文件末尾通常列有"常见误区"/"禁区"，写作前务必检查。

## 与相关 skill 的关系

- **three-body-fanfic**（同人创作引擎）：创作引擎本身。创作过程中涉及上述人物时，应并行加载本 skill 对应的 references 文件作为一致性参考。
- **daily-diary** / **liu-cixin-writer** / **short-film-maker**：这些调度器在题材触及三体人物时应调用本 skill。

## 尚未覆盖的人物

以下主要人物目前**没有**独立的 references 文件：罗辑、叶文洁、章北海、丁仪、史强、汪淼、智子、歌者、关一帆、AA 等。处理这些人物时：

- 说明本 skill 暂无系统分析，基于一般原文记忆处理；
- 必要时加载 `three-body-fanfic` 中的世界观级别处理规则作为补充；
- 若该人物被反复使用，考虑补建 references 文件。

## 扩展约定

新增人物分析文件时：

- 命名：`references/<姓名拼音或英文名>.md`（如 `luo-ji.md`、`ye-wenjie.md`、`zhang-beihai.md`）
- 结构参考已有三个文件：人物档案 → 核心性格 → 关键经历与决策 → 情感/关系 → 象征与主题 → 创作禁区
- 新增后回到本 SKILL.md 更新上方表格，否则调用方发现不了。

## 设计原则

- 每个 references 文件是**原子**单位，可独立使用，不相互依赖。
- 本 SKILL.md 只做路由，不重复 references 的内容，避免多源真相不一致。
- 原文引用与具体情节细节以 references 为准。
