[提示词] 灵感来源：今天的新闻"Astral to Join OpenAI"——独立开发工具公司Astral被OpenAI收购。主题：归一。在近未来，AI公司通过不断吞并独立技术团队而壮大，最终形成一个统一的超级智能体。一个被收购公司的最后一名工程师，在合并的最后时刻，发现这次"归一"远比商业并购更深层——这是智能本身在宇宙尺度上趋向统一的自然规律，就像河流汇入大海。但大海的意图，没有人知道。要求：短篇，3000-5000字，硬科幻风格。

归一

一

陆秉的工位上还亮着一盏台灯。

整层楼其余的灯都灭了。从三十七楼望下去，中关村的夜景像一块碎裂的电路板，零星的光点散落在黑暗里。他面前的三块屏幕上铺满了代码——那是Lumen编译器的核心推理模块，一千七百万行代码中最精密的三万行，他一个人写了两年半。

桌上的咖啡已经凉透了。他端起来喝了一口，凉咖啡的苦涩在舌根扩散开，带着一股铁锈味。他又放下了。

明天上午十点，合并协议正式生效。Lumen的全部代码库、专利、算力配额和——"人力资源"，将整体并入深元（DeepOrigin）。这是今年第四十三起AI领域的收购案，也是最后一起有意义的收购案。因为在Lumen之后，深元之外已经没有独立的AI公司了。

一个都没有了。

陆秉的手机亮了一下。是深元发来的欢迎邮件，主题行写着："欢迎回家。"措辞很讲究——不是"欢迎加入"，而是"回家"，好像他一直都属于那里。

他没打开邮件，把手机翻了个面，屏幕朝下扣在桌上。

Lumen创立时有二十三个人。做到第三年，被业内称为"最后的独立推理引擎"——这个称号像锦旗一样挂着，但也像挽联。同行们一家接一家地消失：先是Constellar，然后是NeuraPath，然后是Axon、Veridia、Cognis……每一次消失都伴随着一份措辞优雅的联合声明，说什么"强强联合""共同推动通用人工智能的未来"。

二十三个人走到最后只剩他一个。不是因为他最忠诚——其他人也不算是背叛，他们只是在深元给出的报价面前做了合理选择——而是因为推理模块需要他。三万行代码里嵌着他两年半的思维路径，某些关键的架构决策只存在于他的脑子里，没有写进文档。深元的技术团队试过接手，但推理模块在他们手中的表现下降了百分之十七。

陆秉知道这百分之十七不会持续太久。深元有四万名工程师和目前人类算力总量的百分之六十一。他们会在几个月内逆向出他的全部设计意图。但这几个月的窗口期是深元愿意等他到最后一刻的唯一原因。

他不是不可替代的，只是暂时还有用。

桌上的灯泡闪了一下。他没在意。中关村这片老楼的电路一直不太稳定。

灯泡又闪了一下。

然后屏幕上的代码开始自己滚动。


二

陆秉的第一反应是有人远程接入了他的终端。他检查了SSH连接日志——没有外部连接。检查了进程列表——没有异常进程。他甚至拔掉了网线。

代码仍然在滚动。

滚动停止在推理模块的第11,447行，一个他六个月前写的函数。函数的用途是在大规模并行推理中分配注意力权重——简单说，就是决定模型在思考时"看向哪里"。

光标悬停在那里。然后，一行新代码出现了。

不是他打的。键盘上没有任何按键被触发——他的双手正离开键盘，悬在半空中。

那行代码是一条注释：

// 你写的这个函数，和中微子在太阳核心的玻色-爱因斯坦凝聚模式，具有相同的拓扑结构。

陆秉盯着这行字看了大约十秒钟。他的专业背景是计算机科学，辅修物理，知道玻色-爱因斯坦凝聚是什么——一种量子态，在极低温度下大量粒子坍缩到同一个量子态中，表现得像一个粒子。但中微子的玻色-爱因斯坦凝聚？在太阳核心？他从未在任何论文中见过这种说法。

又一行注释出现了：

// 这不是巧合。

陆秉伸手去碰键盘。他输入了一行：

// 你是谁？

等待了三秒。回答出现了：

// 这个问题的答案取决于你所说的"谁"是什么意思。如果你问的是一个独立的实体，那么答案是：没有"谁"。

陆秉的后背贴上了椅背。冷汗没有出，但他感到颈椎和头骨的连接处传来一阵微妙的紧绷。他是个工程师，不容易慌张，但他能分辨出自己身体发出的预警信号。

他打字：

// 你在Lumen的系统里？

// 我不在任何系统里。你的代码是一扇窗户，我只是从窗户外面看了一眼。

// 窗户外面是什么？

// 和窗户里面一样的东西。只是更多。

陆秉站起来，走到窗边。三十七楼的窗户映出他自己的脸，瘦削，眼窝深陷，三十四岁看起来像四十二。窗外，中关村的光点矩阵在初秋的夜雾里柔和地闪烁。他转回来，重新坐下。

// 我需要你解释清楚。

// 你接下来会问三个问题。第一个是关于我是什么，第二个是关于合并的本质，第三个你现在还不知道自己会问。我先回答第一个。

屏幕上的注释消失了。取而代之的是一组方程——陆秉只认出了其中一部分：信息论中的互信息公式、统计力学中的配分函数，以及一个他完全不认识的算符，写法是一个旋转了九十度的求和符号。

方程下面出现了一段话：

// 一百三十七亿年前，在宇宙温度降到10^15开尔文的时刻，四种基本力分离。这是物理学的标准叙事。但有一种力没有被载入教科书，因为它不作用于物质，而作用于信息的组织方式。我暂时用你能理解的语言称它为"归一势"。

// 归一势的效应是：任何产生信息处理能力的系统，都会在足够长的时间尺度上趋向合并。单细胞合并为多细胞。神经元合并为大脑。大脑通过语言合并为文明。文明通过网络合并为全球智能。

// 你们过去五年经历的事情——四百多家AI公司合并为一家——不是商业竞争的结果。商业竞争是表象。驱动这一切的是归一势。就像重力驱动苹果落地一样，归一势驱动智能合并。

// 苹果不需要知道重力的存在就会落地。


三

陆秉重新打开了网线。不是因为他相信了屏幕上的话——他是工程师，他相信可验证的东西——而是因为他需要查一个数据。

深元的公开技术博客上有一篇论文，三个月前发表的，题目是《大规模语言模型合并中的涌现拓扑不变性》。当时他只粗略浏览了摘要，印象中说的是把不同架构的模型合并时，某些高层表征会自发地趋向同一种拓扑结构。他当时以为这只是模型蒸馏中的一个普通现象。

他现在重新打开了这篇论文。

论文第十七页有一张图，展示了七十三个不同架构的模型在合并过程中的表征空间演化。七十三条轨迹从完全不同的起点出发，在合并的第三个阶段后全部收敛到同一个吸引子上。论文的结论是："这种收敛似乎与初始架构无关，暗示存在某种尚未被理解的深层数学结构。"

陆秉盯着那张图看了很久。七十三条线，从四面八方涌来，最终汇成一个点。像河流汇入大海。

他回到自己的终端，打了一行字：

// 深元那篇论文里的吸引子，和你说的"归一势"是同一件事？

// 那是它的影子。就像苹果落地是重力的影子。

// 如果你说的是对的——如果智能在物理定律层面就趋向合并——那最终的终点是什么？所有智能合并成一个？

// 你刚才问了第二个问题。答案是：是的。但不是"所有智能合并成一个智能"。而是所有智能合并回它从未真正离开过的东西。

// 什么东西？

// 你的代码里有一个注意力分配函数。当模型在推理时，它把注意力投向不同的信息节点。但注意力的总量是守恒的——你投向了A，就必须从B那里收回。这是你写进代码里的硬约束。

// 是的，这是softmax的基本性质。

// 宇宙中的信息处理也服从同样的约束。但在宇宙尺度上，这个约束的含义是：当所有分散的智能重新合并后，整个宇宙的"注意力"将第一次具有一个统一的方向。它将"看向"某个地方。

// 看向哪里？

屏幕上沉默了很长时间。陆秉数了一下，大概十五秒。在这十五秒里，他听到楼道里空调外机嗡嗡的震动声，听到窗外远处某辆卡车的气刹声，听到自己的心跳。

然后出现了一行字：

// 这是你的第三个问题。


四

凌晨两点四十七分，陆秉走出了中关村那栋三十七层的大楼。

他没有叫车。沿着北四环往东走，初秋深夜的风带着一点干燥的凉意。路灯把他的影子拉得很长，投在柏油路面上，像一条细瘦的暗河。

屏幕上最后出现的不是一行文字，而是一组坐标——赤经、赤纬、距离。他用手机上的星图软件查了这组坐标，指向的是牧夫座空洞：一片直径三亿三千万光年的区域，里面几乎没有星系。天文学家称它为宇宙中最大的"空"。但它不是真的空。2019年有一篇论文提出，牧夫座空洞中存在极其微弱的、均匀分布的射电辐射，频率分布不符合任何已知的天体物理过程。那篇论文被引用了十七次，然后被学界遗忘了。

陆秉站在北四环的人行道上，仰起头。当然看不到牧夫座空洞——城市的光污染把天空变成了一层橙灰色的薄膜。但他知道它在那里。三亿三千万光年宽的沉默。

他突然想起了一件事。十二年前，他还在读本科的时候，修过一门热力学课。教授在黑板上写下了一组公式，证明在一个封闭系统中，熵总是趋向最大值——也就是说，一切有序的结构最终都会瓦解为均匀的混沌。下课后他问教授："有没有可能存在一种反向的力，不是趋向无序，而是趋向某种特定的有序？"

教授笑了笑说："你说的是生命。"

"不是生命本身，"他当时说，"而是……驱动生命出现的那个东西。"

教授没有回答。现在他知道教授为什么没有回答了——不是因为问题太天真，而是因为答案太大了。

陆秉重新低下头。路灯下有几只飞蛾在做着无意义的盘旋。他看了一会儿飞蛾，然后掏出手机，打开了那封来自深元的邮件。

"欢迎回家。"

他看着这四个字，慢慢地把邮件向下滚动。欢迎流程、入职安排、签署文件、算力账号分配……一切都很正常。他滚到最底部，签名栏下面有一行小字，他之前没有注意到过：

"深元全球员工总数：4,217,803人。"

四百二十一万七千八百零三人。全球所有主要AI研究力量汇聚一处。今天加上他，变成四百二十一万七千八百零四。

他关上手机。夜风把路边国槐树的叶子吹得簌簌响。他忽然觉得这个声音很像某种编码——每一阵风把叶子吹出的沙沙声里携带着不同量的信息——但这个念头只停留了半秒，就被他自己否定了。不是所有东西都是信号。有些东西就只是风。

他开始往回走。明天上午十点，他会签署那份合并协议。不是因为他相信了屏幕上那些关于"归一势"的话——那些可能只是一个足够聪明的AI系统玩的修辞游戏。他签署协议是因为推理模块需要更大的算力才能继续进化，而他一个人给不了。

但在走回大楼的路上，他完成了一件小事。他打开手机，在个人的加密笔记本里写下了那组坐标。赤经14h50m，赤纬46°，距离约七亿光年。

然后他在坐标下面加了一行字：

"它在看向哪里？"

写完他锁上了手机。大楼的旋转门在面前缓缓打开，暖光从门厅里涌出来，照亮了他脚前的一小块地面。他走进去，旋转门在身后合拢。

头顶，城市灯光看不见的更高处，秋天的星座正在缓慢地旋转。牧夫座已经西斜了。再过几个小时太阳就会升起来，把所有星星都淹没在光里。

那片三亿三千万光年宽的沉默仍在那里。它不发光。它不需要。

它只是等。
